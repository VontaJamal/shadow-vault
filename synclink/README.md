SyncLink is a private video library extension that helps you save, organize, and watch videos from across the web, with local-first storage.

---

## What It Does

Save videos in one click, organize your library, and find anything quickly with search and filtering.

- **One-click save** ‚Äî inline save on verified sites, manual/context-menu save everywhere
- **Smart library** ‚Äî playlists, tags, full-text search, multiple sort options
- **Privacy-first** ‚Äî all data stored locally, nothing leaves your browser
- **Free + paid tiers** ‚Äî core features free, advanced filtering and organization on paid plans

## Who It's For

- Developers and researchers collecting tutorials, references, and demos
- Learners building personal study libraries from video sources
- Privacy-conscious users who want local control over saved media

## Current State

- Chrome extension foundation is live in the public repository
- Core save, playlist, tagging, and search experiences are in place
- Product polish and distribution work are active
